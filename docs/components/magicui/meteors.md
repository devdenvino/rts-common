# Meteors

Animated meteor shower background effect.

## Usage

```typescript
import { Meteors } from '@devdenvino/rts-common/components/magicui';

<div className="relative">
  <Meteors number={20} />
  <div className="relative z-10">
    Your content
  </div>
</div>
```

## Props

- `number`: Number of meteors (default: 20)
