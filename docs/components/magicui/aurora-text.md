# Aurora Text

Animated gradient text effect.

## Usage

```typescript
import { AuroraText } from '@devdenvino/rts-common/components/magicui';

<AuroraText>Beautiful Gradient Text</AuroraText>
```

## Example

```typescript
<div className="text-center">
  <AuroraText className="text-6xl font-bold">
    Welcome to Our App
  </AuroraText>
</div>
```
